JavaScript könyvtár mátrixokat használó algoritmusok interaktív vizualizációinak készítésére

Szerző: Bc. Ruzsik Márton László

GitHub Repozitórium Dokumentáció (Wiki)
Szakdolgozat Leírása

A szerző feladata egy olyan JavaScript könyvtár létrehozása, amely segítségével könnyen kialakíthatók mátrixokat használó algoritmusok interaktív vizualizációi. A könyvtár segítségével kialakított vizualizációk szemléltessék a megadott méretű mátrixokat, a sor- és oszlopindexek ciklusváltozóit, megjelenítendő programkódokat. A mátrixelemek értékét a vizualizáció lejátszása előtt vagy közben is lehessen módosítani. A vizualizációkat lehessen lépésenként vagy folyamatosan is lejátszani. A lejátszás során a megjelenített programkódban emelje ki az éppen szemléltetett programsort vagy sorokat. A JavaScript könyvtár a teljes vizualizációt a HTML dokumentum Canvas elemén valósítsa meg. A könyvtár elkészítéséhez objektum-orientált programozást használjon, hozzon létre pl. külön osztályokat a vizualizáció vezérlőire, mátrix elemeire, mátrixra, ciklusváltozókra, megjelenített programkódra, stb. Mutassa be az elkészített JavaScript könyvtár használatát néhány példa segítségével.
